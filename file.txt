Есть два вида циклов:

С ПредУсловием: Условие (ромб на блок схеме), действие (прямоугольник на блок схеме), условие (ромб на блок схеме)



С ПостУсловием: Действие (прямоугольник на блок схеме), условите (ромб на блок схеме), действие (прямоугольник на блок схеме).


Цикл со счетчиком обозначается на блок схеме шести угольником.

Сохранить изменения во всей папке (во всех файлах git add .)
